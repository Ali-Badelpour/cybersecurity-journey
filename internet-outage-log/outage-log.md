# Internet Outage & Offline Work Log

**Country:** Iran  
**Reason:** Tehran was attacked on 28 February 2026  

**Period without open internet access:**  
28 February 2026 – 19 April 2026

During this period, only the national/governmental internet was available. Most foreign websites including GitHub, Python documentation, and other learning resources were completely inaccessible.

**From 19 April 2026:**  
On 19 April 2026, open internet remained filtered, but GitHub started working via browser. However, the Git protocol (push/pull) was still inaccessible from the terminal.

**Current status:**  
I have checked again, and Git is now working properly. This log file is being pushed using `git push` normally.

**What I did during the outage:**  
I continued studying cybersecurity from books and completed exercises entirely offline. All code, notes, and screenshots were saved locally with original timestamps.

**Current internet situation:**  
The open internet is still inaccessible. However, at least GitHub, Git, and python.org are working now. If they stop working as well, I will continue working offline again and add another log file.

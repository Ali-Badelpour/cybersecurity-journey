# My Cybersecurity Labs

I document projects I build while learning Linux security, ethical hacking, and network defense. All work is done **on my own devices** (personal laptop, home router, isolated VMs) for educational purposes only.
## Ethical Commitment

I never test techniques on networks or devices I do not own or have explicit permission to assess. These labs help me understand how attacks work so I can better defend against them. Misuse of these skills is illegal and unethical – this repository is solely for learning.
